# From-The-Git-Go

cli tool that will allow you to easily make new repos in github but written in Go...so it's cooler

first:
    `$go build main.go`

then to run:
    `$ ./main -u $GITHUB_USERNAME -p $NEW_GITHUB_REPO_NAME`
